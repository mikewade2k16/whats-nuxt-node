# Frontend do módulo `fila-atendimento`

Este diretório será o destino do frontend plugável do módulo.

## Fonte atual

- `incubadora/fila-atendimento/web/app/features/operation`
- páginas de `operacao`, `relatorios`, `ranking`, `dados`, `inteligencia`
- páginas e stores de `configuracoes` e `consultor`

## Regra

O frontend do módulo deve entrar no host via registry de rotas, menus e capacidades, e não como painel paralelo permanente.
